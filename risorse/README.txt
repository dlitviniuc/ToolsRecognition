In questa cartella ci sono le risorse usate in fase di classificazione

- "foto" contiene un gruppo di immagini non presenti nel dataset, utilizzate per testare la classificazione

- "ClassificationLearnerSession" è una sessione del Classification Learner dove è possibile vedere il metodo usato per generare il classificatore ensamble

- "Classificatori.jpg" è un'immagine estratta da una sessione di Classification Learner mostrante l'accuratezza per la validation e testing dei classificatori possibili

- "comb6" è l'insieme dei tre descrittori su tutto il dataset, viene usato per fare il traing del classificatore

- "ConfusioneValidation" è la matrice di confusione sulla validation del classificatore bagged trees

- "images.list" è la lista con i nomi delle immagini del dataset

- "labels.list" è la lista ordinata come images.list con la ground truth del dataset, quindi l'identità dei vari oggetti

- "lbp" è un file contenente i descrittori lbp per il dataset

- "pers" è un file contenente i descrittori personalizzati per il dataset

- "Pipeline" è l'immagine della pipelie per lo sviluppo del progetto

- "qhist" è un file contenente i descrittori qhist per il dataset

- "Test" è la matrice di confusione sul test a 10% del dataset del classificatore bagged trees

- "trt2" è la struttura contenente il classificatore bagged trees, generato da ProvaClassificazione
